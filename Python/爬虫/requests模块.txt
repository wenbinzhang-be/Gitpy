requests模块：python中原生的一款基于网络请求的模块，功能非常强大，简单便捷，效率极高。
作用：模拟浏览器发请求

如何使用： <requests 模块的编码流程>
    - 指定url
    - 发送请求
    - 获取响应数据
    - 持久化存储

环境安装：
    pip install requests
